### TASK 1: Photo Upload API endpoints (plugmenow-api)

Στο plugmenow-api/ πρόσθεσε photo upload + serving:

1. Εγκατέστησε (αν δεν υπάρχει ήδη): multer, sharp

2. Στο src/routes/photos.js πρόσθεσε/ενημέρωσε:
   - POST /stations/:id/photos (auth required)
     * Accept: multipart/form-data, field name "photo"
     * Max file size: 5MB
     * Accept: image/jpeg, image/png, image/webp
     * Resize με sharp: max width 1200px, quality 85%
     * Δημιούργησε thumbnail: 300px width
     * Αποθήκευση: /data/photos/{station_id}/{uuid}.jpg + {uuid}_thumb.jpg
     * Insert στο photos table
     * Return: { id, url, thumbnailUrl, caption }

   - GET /stations/:id/photos
     * Return: array of { id, url, thumbnailUrl, caption, userName, createdAt }
     * Join με users table
     * Sorted by createdAt DESC

   - DELETE /photos/:id (auth required, only own photos)

3. Πρόσθεσε static file serving για photos:
   - GET /photos/{station_id}/{filename}
   - Serve από /data/photos/
   - Τα photos θα είναι accessible μέσω: /api/photos/{station_id}/{filename}

4. Validation: max 10 photos ανά user ανά station

5. Δοκίμασε τοπικά: npm start, test upload με curl

### TASK 3: Station Data Refresh Script (n8n-ready)

Δημιούργησε ένα self-contained script που μπορεί να τρέξει στον Contabo
χωρίς εξάρτηση από το dev machine:

1. Αντέγραψε/προσάρμοσε scripts/fetch-stations.js → scripts/refresh-stations.js
   - Πρέπει να τρέχει standalone (χωρίς npm run dev)
   - Fetch ΜΥΦΑΗ data
   - Merge με address-cache.json
   - Output: stations.json
   - ΣΗΜΕΙΩΣΗ: Μην αλλάξεις τον υπάρχοντα fetch-stations.js

2. Δημιούργησε scripts/deploy-to-plesk.sh:
   ```
   #!/bin/bash
   # Build + deploy stations data to Plesk
   cd /path/to/project
   node scripts/refresh-stations.js
   # rsync ή scp μόνο τα data αρχεία στο Plesk
   rsync -avz public/data/stations.json user@plesk-server:/path/to/chargegr.viralev.gr/data/
   ```

3. Δημιούργησε n8n workflow JSON (scripts/n8n-refresh-workflow.json):
   - Trigger: Cron κάθε 6 ώρες (0 */6 * * *)
   - Execute: bash scripts/deploy-to-plesk.sh
   - On error: HTTP webhook notification ή email
   - ΣΗΜΑΝΤΙΚΟ: Δομή JSON συμβατή με n8n import

### TASK 4: Vehicle Admin Panel (API endpoints)

Στο plugmenow-api/ πρόσθεσε admin endpoints:

1. Middleware: isAdmin check (role='admin' στο users table)

2. Endpoints:
   - GET /admin/vehicles — λίστα vehicles (pagination, search)
   - POST /admin/vehicles — add vehicle
     body: { brand, model, batteryCapacity, maxChargingPower, connectors }
   - PUT /admin/vehicles/:id — edit vehicle
   - DELETE /admin/vehicles/:id — delete vehicle

3. Vehicle suggestions (community):
   - GET /admin/suggestions — pending suggestions
   - POST /admin/suggestions/:id/approve — approve → add to vehicles
   - POST /admin/suggestions/:id/reject — reject

4. Endpoints για users (non-admin):
   - POST /vehicles/suggest — submit vehicle suggestion
     body: { brand, model, batteryKwh, maxChargingKw, connectorType }
   - GET /vehicles — get all vehicles (public, cached)
   - GET /vehicles/brands — get brands list (public, cached)

5. Export endpoint:
   - GET /admin/vehicles/export — download JSON (format compatible
     με public/data/vehicles.json)

ΣΗΜΕΙΩΣΗ: Τα vehicles τώρα φορτώνονται από static JSON.
Στο μέλλον θα φορτώνονται από το API, αλλά αυτή η αλλαγή
στο frontend ΔΕΝ γίνεται τώρα — μόνο τα API endpoints.
