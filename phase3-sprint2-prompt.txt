## BACKEND API (ήδη τρέχει)
- Base URL: /api (reverse proxy μέσω Plesk → Contabo:3002)
- Health check: GET /api/health → {"status":"ok"}
- Auth: Google Sign-In → POST /api/auth/google → JWT token
- Endpoints:
  - POST /api/auth/google — body: { token } → { jwt, user }
  - GET /api/auth/me — header: Authorization: Bearer <jwt>
  - GET /api/stations/:id/reviews → { reviews, avgRating, count }
  - POST /api/stations/:id/reviews — body: { rating, comment, wasWorking, waitTimeMinutes }
  - GET /api/stations/:id/checkins → { checkins }
  - POST /api/stations/:id/checkins — body: { wasWorking, connectorUsed, chargingSpeedKw, comment }
  - GET /api/user/favorites → { favorites: [stationId, ...] }
  - POST /api/user/favorites/:stationId → { isFavorite: true/false }
  - GET /api/stations/:id/score → { reliabilityPct, totalCheckins, avgRating, totalReviews }

1. Δημιούργησε src/lib/api.ts:

   - Wrapper functions: get(), post(), del() με JWT header

2. Δημιούργησε auth section στο Zustand store (src/store/appStore.ts):

   - Κατά το load: αν υπάρχει jwt, κάνε GET /api/auth/me για validate

3. Δημιούργησε src/lib/auth.ts:
