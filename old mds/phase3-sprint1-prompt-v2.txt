### TASK 3: Auth Endpoints
Υλοποίησε:
1. POST /auth/google
   - Body: { token: "google-id-token" }
   - Verify token με google-auth-library
   - Αν ο χρήστης δεν υπάρχει → create στο users table
   - Αν υπάρχει → update last_login
   - Return: { jwt: "...", user: { id, email, displayName, avatar } }

2. GET /auth/me
   - Header: Authorization: Bearer <jwt>
   - Return: current user profile

3. Auth middleware (middleware/auth.js):
   - Verify JWT
   - Attach user to req.user
   - 401 αν invalid/expired

### TASK 4: Reviews Endpoints
Υλοποίησε:
1. GET /stations/:id/reviews
   - Return: reviews array + avg_rating + count
   - Join με users για display_name, avatar
   - Sorted by created_at DESC

2. POST /stations/:id/reviews (auth required)
   - Body: { rating, comment, wasWorking, waitTimeMinutes }
   - Validation: rating 1-5, comment max 1000 chars
   - Upsert: ένα review ανά user ανά station

### TASK 5: Check-ins Endpoints
Υλοποίησε:
1. GET /stations/:id/checkins
   - Return: recent check-ins (last 30 days)
   - Join με users

2. POST /stations/:id/checkins (auth required)
   - Body: { wasWorking, connectorUsed, chargingSpeedKw, comment }
   - Rate limit: max 1 per user per station per 4 hours

### TASK 6: Favorites Endpoints
Υλοποίησε:
1. GET /user/favorites (auth required)
   - Return: array of station_ids

2. POST /user/favorites/:stationId (auth required)
   - Toggle: add αν δεν υπάρχει, remove αν υπάρχει
   - Return: { isFavorite: true/false }

### TASK 7: Score Endpoint
Υλοποίησε:
1. GET /stations/:id/score
   - Return: { reliability_pct, total_checkins, avg_rating,
     total_reviews, last_checkin }
   - Από τα views station_reliability + station_ratings
   - Cache result σε Redis (TTL 5 λεπτά)
