Στο plugmenow-api project (/media/eleftheria/DataSSD/Projects/ChargeGr/plugmenow-api), πρόσθεσε email/password auth endpoints. Εγκατέστησε bcrypt: npm install bcryptjs. Στο src/routes/auth.js πρόσθεσε: POST /register — body { email, password, displayName }, hash password με bcrypt (10 rounds), insert στο users table, return { jwt, user }. POST /login — body { email, password }, βρες user by email, compare password με bcrypt, return { jwt, user }. Validation: email required + valid format, password min 8 chars, displayName required στο register. Errors: 400 αν λείπουν πεδία, 409 αν το email υπάρχει ήδη, 401 αν λάθος credentials. Πρόσθεσε password_hash VARCHAR(255) column στο users table (ALTER TABLE στο schema). Δοκίμασε με npm start. Git commit.
